1.StayX – Airbnb-Inspired Full-Stack Web Application
Tech Stack          How to use=> npm install, and nodemon app.js                    

• Node.js, Express.js
• MongoDB (Mongoose ORM)
• EJS (Server-side templating)
• Cloudinary (Image storage)
• REST APIs
• Bootstrap 5, CSS


2.Key Features
•  Secure User Authentication
• Password hashing using bcrypt
• Session-based login/logout system
• Protected routes for logged-in users only


3. Full CRUD Functionality
• Create new property listings
• Edit, update, delete listings
• View detailed listing pages with images

4. Cloud Image Upload
• Integrated Cloudinary for image hosting
• Multer for file upload handling

5.Smart Search System
• Location-based dynamic search
• Filter listings by category (mountain, sea, trending, rooms etc.)


6. Review System
• Users can add reviews
• Edit/Delete reviews (owner-only access)

7. Additional Highlights
• Implemented Flash Messages for alerts
• Modular code structure (MVC pattern)
